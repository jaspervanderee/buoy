    Company description

    Phoenix is designed to provide a simple, self-custodial experience for fast, low-cost Lightning transactions directly on mobile devices, emphasizing ease of use for beginners.

    Phoenix is designed to provide a simple, self-custodial experience for fast, low-cost Lightning transactions directly on mobile devices, emphasizing ease of use for beginners.

    Phoenix runs a native Lightning node on the user's device, connecting to ACINQ's node for automated channel management, eliminating the need for swaps and reducing fees for direct Lightning payments. Additionally, Phoenix allows connections to a user's own Bitcoin node for enhanced privacy.

    For beginners, Phoenix's intuitive UI, low-cost Lightning transactions, and self-custodial design make it ideal for those starting with Bitcoin, though channel size limitations may require occasional fee adjustments for frequent receivers. Compared to Breez's merchant-centric features or Aqua's Liquid-focused versatility, Phoenix excels in delivering a straightforward, cost-effective Lightning experience with robust privacy options.

    FAQs

    What is Phoenix Wallet?

    Phoenix is a self‑custodial mobile wallet that runs a Lightning node on‑device for fast payments.

    Does Phoenix support on‑chain?

    Yes. You can send and receive on‑chain using swaps; channel creation can incur higher on‑chain fees.

    Can Phoenix connect to my own node?

    Yes. Phoenix can connect to your own node for improved privacy.
