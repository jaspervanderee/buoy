    FAQs

    Does Umbrel install Bitcoin by default or through its App Store?

    Through the App Store—Umbrel lets you one-click install Bitcoin/Lightning and other apps from a curated store.

    What Lightning implementation does myNode use?

    myNode uses LND and bundles many node tools behind a polished web UI.

    Can I run Umbrel and myNode on Raspberry Pi or x86?

    Yes—both support Raspberry Pi and x86; Umbrel also offers Umbrel Home, and myNode sells prebuilt devices.
